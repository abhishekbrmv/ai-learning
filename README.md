# ai-learning

I'm documenting my journey into AI engineering here. Every folder is a topic I worked through with real, runnable code — not notes or summaries.

The goal is simple: understand how these systems actually work, then build them from scratch.

## What's here

| Folder | What I built |
|--------|--------------|
| `rag-fundamentals` | Embeddings, vector search, chunking, full RAG pipeline |

Adding more every week — agents, fine-tuning, deployment.

## Setup

```bash
git clone https://github.com/your-username/ai-learning.git
cd ai-learning
python3 -m venv venv
source venv/bin/activate
```

Add a `.env` file with your keys:

```
OPENAI_API_KEY=your-key-here
```
